## Brukerne skal forstå hva de har rett til

- Vi skal bruke ord som alle kan forstå.
- Brukerne skal ikke ha behov for å kjenne til juridiske fagord eller NAVs begreper for å forstå tekstene på nav.no.
- Brukerne skal ikke trenge hjelp eller veiledning for å forstå innholdet.

## Brukerne skal raskt finne informasjon som er viktig for dem

- Vi skal starte med informasjonen som er viktigst for brukeren. Det vil si informasjonen som krever handling og som gjør at de raskt kan løse problemet sitt.

## Teksten må skape trygghet

- For å gjøre brukerne selvstendige og selvhjulpne på nav.no må vi bruke et språk som ikke skaper usikkerhet og tvil. Vi må derfor skrive enkelt, tydelig og konkret.
