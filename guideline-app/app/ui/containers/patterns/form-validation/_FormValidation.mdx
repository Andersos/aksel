# Skjemavalidering

<Ingress>Dette mønsteret skal demonstrere reglene for skjema-validering.</Ingress>

## Retningslinjer

- Validering per felt skjer ved `blur` og `change`
- Feiloppsummeringsboks vises ved `submit`
- Feil i feiloppsummeringsboks forsvinner etterhvert som de blir utbedret
- Hvis alle feil i feiloppsummeringsboks blir utbedret, og nye feil blir introdusert vil feiloppsummeringsboksen igjen vises før nytt `submit` forsøk
