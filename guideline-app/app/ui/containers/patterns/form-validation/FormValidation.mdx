# Skjemavalidering

<Ingress>Dette mønsteret skal demonstrere reglene for skjema-validering.</Ingress>

## Retningslinjer

- Validering per felt skjer ikke før etter første `submit` forsøk,  deretter på `change`
- Feiloppsummeringsboks vises ved `submit`
- Feil i feiloppsummeringsboks blir markert som **fikset** etterhvert som de blir utbedret
