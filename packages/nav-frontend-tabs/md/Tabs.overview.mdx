## Kombinasjon med panel

Komponenten passer fint i kombinasjon med [Panel](/components/panel), hvor normal variant bør plasseres
utenfor og langs øverste kant av panelet.

<Alertstripe type="info">
  Inntil videre må du manuelt håndtere visning av riktig panel-innhold når aktiv
  tab endres. Du må også skru av
  <InlineCode className="inline">border</InlineCode> og <InlineCode className="inline">
    border-radius
  </InlineCode> langs toppen av panelet. Vi jobber med å lage en mer avansert <InlineCode className="inline">
    TabPanel
  </InlineCode> komponent som er tenkt å både fikse disse stilene for deg, og håndtere
  automatisk visning av riktig panel-innhold avhengig av aktiv tab.
</Alertstripe>

<Example>
  <Tabs
    tabs={[{ label: "Første" }, { label: "Andre" }, { label: "Tredje" }]}
    onChange={() => {}}
  />
  <Panel
    border
    style={{ borderTop: 0, borderTopLeftRadius: 0, borderTopRightRadius: 0 }}
  >
    Innhold her.
  </Panel>
</Example>

```jsx
<Tabs
    tabs={[
        {"label": "Første"},
        {"label": "Andre"},
        {"label": "Tredje"}
    ]}
    onChange={() => {}}
/>
<Panel border style={{borderTop:0, borderTopLeftRadius: 0, borderTopRightRadius: 0}}>
    Innhold her.
</Panel>
```
